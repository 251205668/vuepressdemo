```markdown

State management：Vuex
Front-end routing：Vue Router
Server communication：axios、jsonp
Mobile terminal scrolling Library：better-scroll
Building tools：webpack 2.0
Source code：es6
```

### The basic component library that this project contains
**Basic components**
```bash
1.confirm：Confirmation dialog components
2.listview：Address list components
3.loading：Loaded components
4.no-result：No result display component
5.progress-bar：Progress bar components
6.progress-circle：Round progress bar component
7.scroll：Rolling component of mobile terminal
8.search-box：Search box components
9.search-list：Search list components
10.slider：Carousel components
11.switches：Switch switching assembly
12.top-tip：Top message prompt component
1.  song-list：Song list component
```
**Business component**
```bash
1.add-song：Add songs to list component
2. Disc: song list detail page component
3. M-header: page header component
4. Music List: song list page component
5. Player: player kernel component
6. Playlist: playlist component
7. Rank: leaderboard page component
8. Recommended: recommended page component
9. Search: search page component
10.Singer: singer page component
11. Singer detail: singer detail page component
12. Suggest: search prompt list component
13. Tab: top navigation bar component
14. Top list: leaderboard detail page component
15. User center: user center page component
```
